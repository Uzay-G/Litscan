Project Description

The Process

This tool analyses a phrase entered by the user and returns with the list of literary techniques his phrase contained. This facilitates the detection of techniques in literary passages or phrases.


In this example from the novel the "Sun Also Rises" by Ernest Hemingway, a simile is used to explicitly compare the cafe and the image of a battleship. This tool does not yet support all techniques and I am trying to enhance its detection and accuracy. Check here to see which techniques can be scanned.


Warning

I cannot devote much time to this project and there are still problems and techniques that cannot be detected. If you encounter a bug, please report it down below.

Thank You

I hope you find this tool useful. I created this project to simply help people and I wish it managed to do that for you.
